Information

TDBuilder is an interactive tool designed to create, visualize, and export Thing Descriptions following the W3C Web of Things (WoT) standard.

What can you do with TDBuilder?

  • Create Thing Descriptions from scratch using visual forms.
  • Edit existing TDs in a structured way.
  • Visualize the structure as an interactive graph.
  • Export and validate the result in JSON format.
